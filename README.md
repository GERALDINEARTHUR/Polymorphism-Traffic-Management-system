# Smart Traffic Management System

## About the Project

The Smart Traffic Management System is a beginner-friendly Python project that demonstrates the application of Object-Oriented Programming (OOP) concepts. It models several traffic control devices that inherit from a common base class while exhibiting different behaviors through polymorphism.

This project emphasizes writing clean, reusable, and scalable code using inheritance and method overriding.

## Objectives

- Explore the principles of Object-Oriented Programming.
- Demonstrate inheritance using a parent and multiple child classes.
- Apply polymorphism by allowing different objects to respond to the same method.
- Build a simple yet extendable traffic management simulation.

## Class Structure

TrafficDevice
│
├── TrafficLight
├── SpeedCamera
├── PedestrianSignal
└── EmergencySiren

## Class Overview

### TrafficDevice
Serves as the parent class and defines the common activate() method for all traffic devices.

### TrafficLight
Simulates the operation of a traffic signal by displaying a green light.

### SpeedCamera
Represents a traffic monitoring device that detects speeding vehicles.

### PedestrianSignal
Simulates a pedestrian crossing signal that controls safe road crossing.

### EmergencySiren
Represents an emergency alert system that notifies road users of approaching emergency vehicles.

## Key Features

- Simple and easy-to-understand Python code.
- Demonstrates inheritance and code reusability.
- Uses method overriding to customize device behavior.
- Implements polymorphism with a shared activate() method.
- Can be extended by adding more traffic management devices.

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
